内部邮件及论坛类问题

1. 为何我收到新邮件提醒，在内部邮件的收件箱中却打不开新邮件？

答：这是因为您的邮箱空间已满。万户ezOFFICE系统对每个用户的邮箱大小进行了限制，当您的邮箱空间不足时，你将只能看到新邮件的提醒，却打不开新邮件。可以将带附件的历史邮件清理一下，或者是让系统管理员将您的空间大小设置改大一些，这样就可以看到新邮件了。

由于邮件的使用频率非常高，系统运行一段时间后，服务器硬盘就被大量没有保存价值的历史邮件占用了。这样的设计就会让用户定时自行清理自己的邮件，以提高系统空间的利用率。 

2. 如何才能撤回发出的邮件？同一封邮件发给多人，其中一人已经查看，那么其他人收到的邮件还可以撤回吗？

答：只有收件人未打开该邮件的情况下才可以撤回所发的邮件。对于同一封邮件也是这样，同一封邮件发给多人，已经查看的用户的这封邮件不可以撤回，其他未查看的用户的这封邮件可以撤回。

 

3. 为什么有的用户发邮件的时候可以选择一个部门，而我只能选择用户？

答：因为ezOFFICE系统设计了邮件群发的权限，系统管理员没有赋予此权限的用户是无法看到组织前的复选框，也就无法选择整个部门。

4. 关于论坛帖子如何删除？

答：只有论坛的版主可以删除帖子，发帖人自己只能修改自己的帖子。
